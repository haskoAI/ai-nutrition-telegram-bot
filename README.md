# AI Nutrition Telegram Bot (Cal AI)

Интеллектуальный Telegram-бот для анализа питания, подсчёта калорий и генерации рекомендаций с помощью AI.

## 📌 Описание

Бот принимает текст, голос или фото еды и:

- анализирует приём пищи
- рассчитывает калории и БЖУ
- сохраняет данные
- формирует отчёты
- помогает достигать целей

## ⚙️ Как работает

1. Пользователь отправляет:
   - текст
   - голос
   - фото еды

2. Система:
   - преобразует голос в текст
   - анализирует изображение (GPT-4o)
   - обрабатывает запрос через AI-агента

3. AI агент:
   - определяет действие
   - вызывает нужный инструмент
   - формирует ответ

4. Данные сохраняются в Google Sheets

5. Пользователь получает:
   - анализ еды
   - отчёт по калориям
   - рекомендации
